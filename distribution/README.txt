hazel is a minimal content delivery network microservice that powers:

    - maven.floofy.dev
    - cdn.floofy.dev
    - maven.noelware.org
    - cdn.noelware.org

For more information about hazel, you can read the documentation here:
               >- https://docs.floofy.dev/hazel -<

Wondering how hazel was built and distributed? You can check it on GitHub (and star it~):
               >- https://github.com/auguwu/hazel <-

Seen any issues while running hazel? You can always report it on the GitHub repository:
               >- https://github.com/auguwu/hazel/issues -<
